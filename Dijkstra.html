Path Finding using Dijkstra's Algorithm

Programmed in Java

The code presented was part of an assignment I completed for my Algorithms and Data structures course at KTH. The code was written in Java. This was my implementation of a path solving program, using Dijkstras Algorithm. The code was implemented using a queue, Hash table and of course the algorithm itself. You can try out the code by going to the GitHub link below and entering any two european cities as shown. The result will be the shortest time to get from one city to the next.
